TP EJB | BookShop

L' application a été dévelopé sous jboss 5.1 et tourne avec une base de donnée Mysql 5. Elle génére la base de donnée au démarrage du servuer jboss, et la supprime lorsque ce dernier s'arrête. Tous les enregistrements en base seront perdus, une fois le serveur arrêté. Pour commencez, il faut d'abord ajouter des articles via le formulaire prévu à cet effet, et ensuite on pourra faire notre panier. NB: Créer une base nommé bookshop, et bien vérifier les identifiants de connexion à la base dans le fichier bookshop-ds.xml dans le META-INF dans le module EJB.
